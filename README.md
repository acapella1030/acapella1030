
## ☕ Sobre mí

## 🤖 Lo que hago
- 🔧 **Pentesting**: Desafío sistemas buscando sus puntos débiles (con permiso, obvio 😅).

- 🔥 **Respuesta a incidentes**: Si hay un problema de seguridad, me encargo de apagar el fuego.
